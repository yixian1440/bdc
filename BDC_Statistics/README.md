# 不动产登记收件明细统计系统

## 系统功能

本系统专为不动产登记收件管理设计，主要功能包括：

### 核心功能
- **收件登记**：记录收件人、收件类型、收件日期和案件描述
- **统计展示**：按人员、类型、时间周期进行多维统计
- **公平分配**：根据历史数据智能推荐下次收件分配
- **数据管理**：支持数据的增删改查操作

### 统计维度
- **人员统计**：顾永彪、杨苗苗、童昭仪、杨海艺四人的收件情况
- **类型统计**：一般件、开发商、国资、多人分割转让、其他复杂件
- **时间统计**：支持月度、年度和全部数据统计

## 使用方法

### 1. 收件登记
1. 选择收件人（四人之一）
2. 选择收件类型（五种类型之一）
3. 选择收件日期（默认当天）
4. 填写案件描述（可选）
5. 点击"添加收件"按钮保存

### 2. 数据查看
- **统计面板**：查看各类统计数据和图表
- **明细表格**：查看所有收件记录的详细列表
- **时间筛选**：通过顶部筛选器查看不同时间段的数据

### 3. 公平分配建议
系统会根据历史数据自动生成下次收件分配建议，确保工作分配的公平性。

## 技术特点

- **本地存储**：数据保存在浏览器本地，无需服务器
- **响应式设计**：支持电脑和手机访问
- **实时统计**：数据变化时统计结果实时更新
- **颜色标识**：不同类型用不同颜色区分，直观易懂

## 数据安全

所有数据都保存在浏览器的本地存储中，不会上传到任何服务器，确保数据安全。

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）
