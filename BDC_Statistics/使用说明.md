## 📋 主要功能

### 1. 收件录入
- 录入案件基本信息
- 支持多种案件类型
- 自动分配给处理人
- 实时通知

### 2. 案件管理
- 查看我的案件
- 案件状态跟踪
- 处理进度管理

### 3. 统计分析
- 案件类型统计
- 收件人统计
- 开发商统计
- 实时数据更新

### 4. 开发商功能
- 开发商转移案件录入
- 申请人（权利人）必填
- 联系电话非必填
- 实时查看统计

## 🚀 使用步骤

### 登录系统
1. 打开浏览器，访问 http://localhost:5174/login
2. 输入用户名和密码
3. 点击「登录」按钮

### 录入案件（开发商）
1. 登录系统
2. 点击左侧菜单「收件录入」
3. 填写案件信息：
   - 收件编号（必填）
   - 收件类型（默认开发商转移）
   - 收件日期（必填）
   - 申请人（必填）
   - 代理人（必填）
   - 联系电话（非必填）
   - 开发商（必填，选择楼盘）
   - 案件描述（可选）
4. 点击「提交录入」按钮

### 查看统计
1. 登录系统
2. 点击左侧菜单「统计分析」
3. 查看不同维度的统计图表
4. 可选择年份和月份筛选

## ⚠️ 注意事项

1. **开发商转移案件**：
   - 必须填写申请人（权利人）
   - 联系电话为非必填
   - 案件编号可自定义，未填写时自动生成

2. **系统规则**：
   - 录入即视为完成，无需后续处理
   - 案件自动分配给处理人
   - 数据实时更新

3. **浏览器建议**：
   - 推荐使用 Chrome 或 Edge 最新版
   - 关闭浏览器拦截器
   - 允许弹出窗口
